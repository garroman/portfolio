Länge des längsten Substrings finden

16-02-2023

In diesem Beitrag möchte ich kurz über ein Problem auf Leetcode schreiben.

Das Problem besteht darin, eine Funktion zu schreiben, die die Länge des längsten Substring ohne sich wiederholende Zeichen in einem String s zurückgibt.



Beispiele:

Eingabe: s = "abcd"
Ausgabe: 4
Erklärung: Die Antwort ist die gesamte String "abcd", welches eine Länge von 4 hat.



Eingabe: s = "bbbbb"
Ausgabe: 1
Erklärung: Die Antwort ist das String "b", welches eine Länge von 1 hat.



Eingabe: s = "pwwkew"
Ausgabe: 3
Erklärung: Die Antwort ist das String "wke", welches eine Länge von 3 hat.

Lösungen:

Um dieses Problem zu lösen, müssen wir alle maximalen Substrings finden, die keine wiederholdenden Zeichen enthält.

Ich erkläre zunächst zwei Lösungsansätze, die uns dabei helfen können, dieses Ziel zu erreichen.



Erste Lösung:

Für diese Methode werden wir das zugehörige Substring finden, das keine wiederholdenden Zeichen enthält, und zwar für alle Zeichen in der Zeichenkette.

Dazu benötigen wir eine Maximale-Variable erstellen, die wir maxLength nennen werden.

Wir benötigen auch ein sogenannte "outer loop" oder nur einfach eine erste Schleife.
Diese Schleife wird durch jedes Zeichen der Zeichenkette geführt werden.

Und eine "inner loop" oder zweite Schleife hinein die erste.

Ich heiße outerChar das Zeichen, das in der erste Schleife ist.
Und innerChar das Zeichen, das in der zweite Schleife ist.

Wenn outerChar == innerChar, dann sagt das uns, dass wir schon eines wiederholdende Zeichen gefunden haben.

Dann gleichen wir maxLength zu max(maxLength, length)


Code (Java):

int maxLength = 0; for (int i = 0; i < s.length(); i++) {  char outerChar = s.charAt(i)  for (int j = i + 1; j < s.length(); j++) {   char innerChar = s.charAt(j);   int actualLength = j - i + 1   if (innerChar == outerChar) {    maxLength = Math.max(maxLength, actualLength - 1);   }  }  return maxLength; }
