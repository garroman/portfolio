Finding the size of the largest substring

16-02-2023

This post is aimed to explain the solution to a problem I found on Leetcode.

The problem is asking to create a function that retrieves the maximum length of a substring without repeating characters inside a string s.

Examples:

Input: s = “abcd”
Output: 4
Explanation: The answer is the length of the string “abcd”, which has a length of 4.

Input: s = “bbbbb”
Output: 1
Explanation: The answer is the length of the substring “b”, which has a length of 1.

Input: s = “pwwkew”
Output: 3
Explanation: The answer is the length of the substring “wke”, which has a length of 3.

Solutions:

To solve this problem, we must find the maximum length between each non repeating character substring in the string.

For this two solutions will be explained.

Each code snippet is written in Java, that does however not mean that its runnable code, its just a representation.

First solution:

For this method each substring will be evaluated and a variable containing the maximum length that will be called maxLength.

Two loops are required, one for the outer character and one for the inner one.

The inner character will be called as innerChar and the index of these character as right.

The inner loop will be ended when a repeating character is found or the end of the string was reached.

Then the value of the variable maxLength will be updated in case the last substring length was greater than our variable.

In this simple way the goal can be reached.

Code:

int maxLength = 0;
int right = 0;

for (int i = 0; i < s.length(); i++) {
	for (right = i + 1; right < s.length(); right++) { 
		char innerChar = s.charAt(right);
		if (s.indexOf(innerChar) != right) {
			break;
		}
	}
	maxLength = Math.max(maxLength, right - i);
}
return maxLength;

First solution's conclusion:

This solution is enough to solve the problem, but it's a solution which time execution grows exponientally.

Therefore we aim to create another solution that can run through the string once, a linear solution.

Such a solution will be implemented as the second solution to the problem.

Second solution:

The method used to solve the problem in this solution is called the “Sliding Window Technique”.

A "window" or greatest (in lenght) substring will be created and updated whenever a repeating character is found.

And at the end the length of such substring, represented by the variable maxLength will be returned.

For that it is required to create a Hashmap that will be contained in the variable called charMap. This map will help to maintain a record of the index of every character that we have already "seen" or iterated over.

And to know the actual indexes of the substring two variables that will be called left and right are required.

Code:

int maxLength = 0;  
Map<Character, Integer> charIndexMap = new HashMap<>();  
int left = 0;  

for (int right = 0; right < s.length(); right++) {
	Character actualChar = s.charAt(right);
	int index = charIndexMap.getOrDefault(actualChar, -1);
	// contained in substring.
	if (index >= left) {
		left = index + 1;
	} else {
		maxLength = Math.max(maxLength, right - left + 1);
	}
	charIndexMap.put(actualChar, right);
}

return maxLength;
