Länge des längsten Substrings finden

16-02-2023

In diesem Beitrag möchte ich kurz über ein Problem auf Leetcode schreiben.

Das Problem besteht darin, eine Funktion zu schreiben, die die Länge des längsten Substring ohne sich wiederholende Zeichen in einem String s zurückgibt.

Beispiele:

Eingabe: s = “abcd”
Ausgabe: 4
Erklärung: Die Antwort ist das gesamte String “abcd”, welches eine Länge von 4 hat.

Eingabe: s = “bbbbb”
Ausgabe: 1
Erklärung: Die Antwort ist das Substring “b”, welches eine Länge von 1 hat.

Eingabe: s = “pwwkew”
Ausgabe: 3
Erklärung: Die Antwort ist das Substring “wke”, welches eine Länge von 3 hat.

Lösungen:

Um dieses Problem zu lösen, müssen wir alle maximalen Substrings finden, die keine wiederholende Zeichen enthält.

Ich erkläre zunächst zwei Lösungsansätze, die uns dabei helfen können, dieses Ziel zu erreichen.

Alle Codebeispiele sind in Java geschrieben und dienen nur der Veranschaulichung der Lösungsidee.

Erste Lösung:

Für diese Methode werden wir das zugehörige Substring finden, das keine
wiederholenden Zeichen enthält, und zwar für alle Zeichen in der Zeichenkette.

Dazu benötigen wir eine maximale Variable erstellen, die wir maxLength bezeichnen werden.

Wir benötigen auch ein sogenannte “outer loop” oder nur einfach eine erste Schleife.
Diese Schleife wird durch jedes Zeichen der Zeichenkette iterieren.

Und eine zweite Schleife wird in die erste Schleife eingebettet.

Ich bezeichne das aktuelle Zeichen in der zweiten Schleife als innerChar und den index dieses Zeichnen als right.

Wir beenden die zweite Schleife, wenn wir ein Zeichen finden, das bereits im aktuellen Substring enthalten ist, oder wenn wir das Ende der Zeichenkette erreicht haben.

Dann aktualisieren wir die Variable maxLength auf die Länge des letzten Substrings,falls dieser länger ist als der aktuelle Wert von maxLength.

Auf diese Weise können wir am Ende der Zeichenkette die maximale Substring-Länge gefunden und zurückgegeben haben.

Code:

int maxLength = 0;
int right = 0;

for (int i = 0; i < s.length(); i++) {
	for (right = i + 1; right < s.length(); right++) { 
		char innerChar = s.charAt(right);
		if (s.indexOf(innerChar) != right) {
			break;
		}
	}
	maxLength = Math.max(maxLength, right - i);
}
return maxLength;

Abschluss der erste Lösung:

Diese Lösung reicht aus, um das Problem zu lösen, aber das ist oft nicht ausreichend. Die Lösung des Problems ist nur der erste Schritt.

Wir streben Lösungen an, die für unseren Kontext geeignet sind, und in diesem Fall ist diese Lösung zu langsam.

Die Lösung ist langsam, weil sie zwei Schleifen benötigt, um für jedes Zeichen den Rest des Strings durchzulaufen. Um die Geschwindigkeit des Algorithmus zu verbessern, wäre es sehr gut, wenn es nur einmal durch das gesamte String iteriert.

Wir werden diese letzte Alternativ in der zweiten Lösung erkunden.

Zweite Lösung:

Diese nächste Methode nennt sich “Sliding Window Technique”.

Wir werden einen Teil des Strings als längsten Substring behandeln und diesen aktualisieren, sobald wir ein wiederholendes Zeichen finden.

Nach jedem Zeichen werden wir unsere maxLength aktualisieren, falls der aktualisierte Substring länger und noch nicht im Substring enthalten ist.

Dazu werden wir ein HashMap nutzen, das wir als Variable charMap bezeichnen werden. Dieses Map wird uns dabei helfen, einen Datensatz zu haben, über jedes Zeichen die wir schon “gesehen” haben.

Für die Verfolgung des Anfangs und des Endes unseres Substrings benötigen wir auch zwei Zeiger-Variablen die wir left und right nennen werden.

Code:

int maxLength = 0;  
Map<Character, Integer> charIndexMap = new HashMap<>();  
int left = 0;  

for (int right = 0; right < s.length(); right++) {
	Character actualChar = s.charAt(right);
	int index = charIndexMap.getOrDefault(actualChar, -1);
	// im Substring enthaltet.
	if (index >= left) {
		left = index + 1;
	} else {
		maxLength = Math.max(maxLength, right - left + 1);
	}
	charIndexMap.put(actualChar, right);
}

return maxLength;
